搭建Shadowsocks科学上网

2019-11-30

Shadowsocks官网介绍：一个安全的socks5代理，旨在保护您的互联网流量。
由于经常访问Github，Google等国外网站，但是由于不可抗拒原因又访问不了，所以萌生了自己搭建一个科学上网节点的想法，如果你也想这样做，请确保你有一台非中国大陆的服务器。服务器的选择离中国越近延迟越低。

前段时间花了十几块钱嫖了一个AWS的一年云服务器，选的是韩国节点，平常延迟在80左右，很稳定，关键是安全，不用担心信息泄露和节点挂掉！
好啦，接下来开始我的搭建教程，如果你不想动手搭建，我也提供了一个我的节点账号，你可以翻到底部获取。

本搭建基于Ubuntu，其他操作系统可以此为参考。

  1. 安装python环境和pip
    执行以下命令确认自己有没有Python环境。

    python -V

    如果执行返回类似： Python 3.6.8 这种信息你可以跳过这一步！如果没有需要安装Python环境。执行以下命令将帮助你安装。

    sudo apt install python3 python3-pip

  2. 安装ShdowSocks

    pip install git+https://github.com/shadowsocks/shadowsocks.git@master

  3. 创建配置文件，根据需要复制以下内容到config.json

    多用户配置：

    {
    "server":"39.107.118.182",
    "local_address": "127.0.0.1",
    "local_port":1080,

    "port_password": { 
            "6234": "admin1",   
            "6235": "admin3"
    },
    "timeout":300,
    "method":"aes-256-cfb"
}

    单用户配置：

    {
    "server":"39.107.118.182",
    "server_port":6234
    "local_address": "127.0.0.1",
    "local_port":1080,

    "port_password": "123456",
    "timeout":300,
    "method":"aes-256-cfb"
}

    mkdir /etc/shadowsocks
nano config.json

    保存退出命令： Ctrl+O, Ctrl+X

  4. 启动服务

    ssserver -c /etc/shadowsocks/config.json -d start

    显示以下内容表示成功：

    INFO: loading config from /etc/shadowsocks/config.json
2019-11-30 21:51:36 INFO     loading libcrypto from libcrypto.so.1.1
started
