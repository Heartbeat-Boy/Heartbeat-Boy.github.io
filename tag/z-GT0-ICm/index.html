使用GitHub Page + Gridea 搭建不用不用备案的博客
2018-12-12

👏 欢迎进来观看这篇文章 ！
本站就是基于GitHub Page + Gridea 搭建的，域名买了备案还没完成，但是发现绑定自己的域名没有任何问题，就是访问的时候由于是GitHub所以访问速度就那样，后续会迁移到自己服务器上。

GitHub 上的操作

  1. 首先注册一个github账号，这个很简单，只要你拥有一个自己的邮箱，就可以轻松注册下来。
  2. 创建好以后，点击左侧 Repositories 旁边的 new 按钮
